# 🧾 IT Vendor Price Comparison System

A simple and professional web-based system designed to generate vendor price comparison reports in PDF format.

This project was created to ease my daily procurement workflow at **TMS Sdn Bhd**, allowing quick comparison between vendors and automatic generation of a structured PDF report to be sent to senior management.

---

## 🚀 Overview

The **IT Vendor Price Comparison System** helps:

* Compare multiple vendor quotations
* Automatically calculate total prices
* Identify the lowest/best price in real-time
* Generate a clean, professional PDF report
* Standardize internal procurement documentation

This system improves efficiency, reduces manual errors, and ensures consistent reporting format for internal use.

---

## ✨ Features

* ✅ Add unlimited vendor entries
* ✅ Auto calculation (Price × Quantity)
* ✅ Real-time lowest price detection
* ✅ Professional PDF export
* ✅ Clean and responsive UI (TailwindCSS)
* ✅ Uppercase standardization for vendor names
* ✅ Ready-to-send report format

---

## 🖥️ Tech Stack

* **HTML5**
* **Tailwind CSS**
* **Vanilla JavaScript**
* **jsPDF**
* **jsPDF-AutoTable**
* **Lucide Icons**

---

## 📄 How It Works

1. Enter the **Item Name**
2. Select **Prepared By**
3. Add vendors and input:

   * Company Name
   * Price (RM)
   * Quantity
4. System automatically:

   * Calculates total price
   * Highlights the best (lowest) price
5. Click **Export PDF**
6. PDF is generated and ready to be sent
